এই guide এ দেখানো হয়েছে কিভাবে employee salary setup করবেন, payroll process করবেন, এবং salary reports generate করবেন।

timeline Payroll Process Flow

Salary Setup → Attendance → Leave → Overtime → Payroll Calculation → Payment

account_balance_wallet Step 1: Setup Employee Salary

📋 Salary Structure:

Employee Salary তে basic salary, allowances, deductions সব define করা হয়।
HRM → Employee Management → Salaries → Add

Example Salary Structure:

Employee: John Doe
Basic Salary: ৳30,000
House Rent: ৳12,000 (40%)
Medical: ৳3,000
Transport: ৳2,000
Gross Salary: ৳47,000

💰 Common Salary Components:

Basic Salary
মূল বেতন - সাধারণত gross এর 50-60%
House Rent Allowance
বাড়ি ভাড়া - basic এর 40-50%
Medical Allowance
চিকিৎসা ভাতা - fixed amount
Transport Allowance
যাতায়াত ভাতা - fixed amount

remove_circle Step 2: Salary Deductions

📋 Common Deductions:

Tax Deduction
Income tax - as per tax slab
Provident Fund
PF contribution - usually 10% of basic
Loan Deduction
Employee loan installment
Absence Deduction
Unpaid leave বা absent days এর জন্য

calculate Step 3: Payroll Calculation

📋 Calculation Formula:

Net Salary = Gross Salary + Overtime + Bonus - Deductions - Absence

Example Calculation:

Gross Salary: ৳47,000
+ Overtime: ৳2,000
+ Bonus: ৳5,000
- Tax: ৳3,000
- PF: ৳3,000
- Absence: ৳1,000
= Net Salary: ৳47,000
💡 Auto Calculation: System automatically attendance, leave, overtime থেকে salary calculate করে। Manual adjustment শুধু special cases এ করতে হয়।

task_alt Step 4: Monthly Payroll Process

📋 Payroll Steps:

  1. মাস শেষে attendance finalize করুন
  2. সব leave applications approve/reject করুন
  3. Overtime entries verify করুন
  4. Payroll generate করুন (auto calculation)
  5. Individual salary slips review করুন
  6. Adjustments করুন (if needed)
  7. Payroll approve করুন
  8. Bank transfer file generate করুন
  9. Payment process করুন
  10. Salary slips distribute করুন

receipt_long Step 5: Salary Slip Generation

📋 Salary Slip Contents:

✓ Employee details (Name, ID, Department, Designation)
✓ Salary period (Month, Year)
✓ Earnings breakdown (Basic, Allowances, Overtime, Bonus)
✓ Deductions breakdown (Tax, PF, Loans, Absence)
✓ Attendance summary (Present, Absent, Leave, Holidays)
✓ Net payable amount

tips_and_updates Best Practices

✓ Fixed Schedule: প্রতি মাসের একটি fixed date এ payroll process করুন
✓ Verification: Payment করার আগে সব calculations verify করুন
✓ Backup: Payroll data এর regular backup রাখুন
✓ Transparency: Employees কে salary structure clearly বুঝিয়ে দিন
✓ Compliance: Labor law এবং tax regulations follow করুন
