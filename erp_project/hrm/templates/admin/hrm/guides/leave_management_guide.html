timeline Leave Management Flow

Leave Types → Leave Balance → Application → Approval → Attendance Update

category Step 1: Setup Leave Types

📋 Leave Type কি?

Leave Type হলো বিভিন্ন ধরনের ছুটি। যেমন: Casual Leave, Sick Leave, Annual Leave ইত্যাদি।
HRM → Leave Management → Leave Types → Add

Example: Casual Leave

Code: CL
Name: Casual Leave
Days Per Year: 10 days
Carry Forward: Yes (max 5 days)
Requires Approval: Yes ✅
Paid: Yes ✅

📊 Common Leave Types:

Casual Leave (CL)
সাধারণ ছুটি - 10 days/year, paid, carry forward allowed
Sick Leave (SL)
অসুস্থতার ছুটি - 14 days/year, paid, medical certificate required
Annual Leave (AL)
বার্ষিক ছুটি - 18 days/year, paid, advance booking required
Unpaid Leave (UL)
বেতন ছাড়া ছুটি - unlimited, unpaid, special approval required

account_balance_wallet Step 2: Manage Leave Balance

📋 Leave Balance কি?

Leave Balance হলো প্রতিটি employee এর available leave days। Automatically calculate হয় বা manually adjust করা যায়।
HRM → Leave Management → Leave Balances

Example Leave Balance:

Employee: John Doe
Leave Type: Casual Leave
Year: 2024
Total Allocated: 10 days
Used: 3 days
Remaining: 7 days ✅
💡 Auto Allocation: নতুন fiscal year শুরু হলে automatically সব employee এর leave balance allocate হয়ে যায়। Manual adjustment এর দরকার শুধু special cases এ।

pending_actions Step 3: Leave Application Process

📋 Application Process:

HRM → Leave Management → Leave Applications → Add

Example Application:

Employee: John Doe
Leave Type: Casual Leave
From Date: 2024-03-15
To Date: 2024-03-17
Total Days: 3 days
Reason: Personal work
Status: Pending ⏳

📊 Application Status:

⏳ Pending: Application submitted, waiting for approval
✅ Approved: Leave approved, balance deducted
❌ Rejected: Leave rejected, balance not affected
🚫 Cancelled: Employee cancelled the application

approval Step 4: Approval Process

📋 How to Approve/Reject:

Approval Steps:

  1. Go to: HRM → Leave Management → Leave Applications
  2. Filter by Status: Pending
  3. Click on application to view details
  4. Check leave balance and reason
  5. Change Status to "Approved" or "Rejected"
  6. Add approval notes (optional)
  7. Save
💡 Auto Update: Leave approve হলে automatically leave balance deduct হয় এবং attendance record update হয়। Manual intervention এর দরকার নেই।

celebration Step 5: Configure Holidays

📋 Holiday কি?

Holiday হলো public holidays এবং weekly offs। এই দিনগুলোতে attendance mandatory না এবং leave balance affect হয় না।
HRM → Leave Management → Holidays → Add

Example Holiday:

Name: Independence Day
Date: 2024-03-26
Type: Public Holiday
Applies To: All Employees
Paid: Yes ✅

📅 Common Holidays (Bangladesh):

Independence Day
March 26
Victory Day
December 16
Eid-ul-Fitr
Variable (2-3 days)
Eid-ul-Adha
Variable (2-3 days)
