factory Example Company: "Fashion Tex Ltd"

একটি garments manufacturing company যারা T-Shirt, Shirt, Pants তৈরি করে এবং export করে। তাদের 200 জন worker, 5টি production line, এবং monthly turnover ৳50 লাখ।

shopping_cart Type 1: Cost of Goods Sold (COGS) - Purchase Module দিয়ে

📦 Raw Material Purchase (কাঁচামাল কেনা)

Scenario: Fashion Tex Ltd কে 1000 meter fabric কিনতে হবে T-shirt তৈরির জন্য
Step 1: Purchase Order
Purchase → Purchase Orders → Add
  • Supplier: "Fabric Suppliers Ltd"
  • Product: Cotton Fabric
  • Quantity: 1000 meter
  • Rate: ৳150/meter
  • Total: ৳1,50,000
  • Status: Save as "Sent"
Step 2: Goods Receipt
Inventory → Goods Receipt (PO)
  • Select PO: PO-2026-001
  • Received Qty: 1000 meter
  • Warehouse: Main Warehouse
  • Status: "Received"
  • ✅ Stock IN হবে
Step 3: Purchase Invoice
Purchase → Purchase Invoices → Add
  • Link to PO: PO-2026-001
  • Invoice Amount: ৳1,50,000
  • Payment: Bank Transfer
  • Status: "Paid"
  • 💰 COGS Entry হবে
📊 Result: Purchase Invoice এর amount automatically COGS হিসেবে count হবে এবং User POS Dashboard এ দেখাবে।

groups Type 3: Salary & Wages - HRM Module (Payroll)

💰 Monthly Payroll Processing

Scenario: Fashion Tex Ltd এর 200 জন employee এর January 2026 এর salary process করতে হবে
HRM → Payroll → Process Payroll
Steps:
  1. Select Month: January 2026
  2. Select Employees: All (200 employees)
  3. System automatically calculates:
    • Basic Salary
    • Allowances (House Rent, Medical, Transport)
    • Overtime (if any)
    • Deductions (Tax, Provident Fund)
  4. Review & Approve
  5. Process Payment

info ⚠️ গুরুত্বপূর্ণ নোট

  • COGS Entry: Purchase Invoice create করলেই automatically COGS হিসেবে count হবে। আলাদা Journal Entry করার দরকার নেই।
  • Operating Expenses: সব operating expense এর জন্য Finance Module এ Journal Entry করতে হবে।
  • Payroll: HRM Module এ payroll process করলে automatically Finance এ Journal Entry হবে।
  • Dashboard: User POS Dashboard এ শুধু COGS দেখাবে। সব expense দেখতে Financial Dashboard ব্যবহার করুন।
  • Reports: Profit & Loss Report এ সব expense একসাথে দেখা যাবে।
  • Approval: বড় expense এর জন্য manager approval নিতে হবে।
