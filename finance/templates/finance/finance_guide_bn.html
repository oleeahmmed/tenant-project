এন্ড-টু-এন্ড গাইড

এই গাইড এমনভাবে লেখা যে একজন নতুন মালিক, নতুন অ্যাকাউন্ট্যান্ট, এমনকি একদম নতুন user-ও শুধুমাত্র এই পেজ দেখে পুরো Finance সেটআপ ও চালু করতে পারে।

গল্পে আমাদের company: Demo Traders Ltd., মালিক: রাহাত, অ্যাকাউন্ট্যান্ট: তানিয়া।

Part A: প্রথম দিন কী করবেন (Owner checklist)

1) Tenant/Company ready

Owner signup করে company create করবে, admin user নিশ্চিত করবে।

2) Team access

Tenant auth থেকে অ্যাকাউন্ট্যান্ট user add করে proper permission দিন।

3) Foundation master data

Currency, exchange rate, tax, customer, supplier, payment method add করুন।

4) Fiscal plan

কোন মাস থেকে হিসাব শুরু হবে ঠিক করুন (যেমন Jan-Dec fiscal year)।

Part B: Foundation setup details (Finance এর আগে বাধ্যতামূলক)

  • - Currencies: কমপক্ষে BDT active করুন
  • - Exchange rates: foreign currency থাকলে daily/weekly rate দিন
  • - Tax rates: VAT 0%, 5%, 15% (business অনুযায়ী)
  • - Customers: AR invoice এর জন্য দরকার
  • - Suppliers: AP invoice এর জন্য দরকার
  • - Payment methods: Cash / Bank transfer / Mobile

Part C: Finance setup details (zero to live)

  1. 1) Chart of Accounts তৈরি
  2. 2) Fiscal year create
  3. 3) Fiscal periods (১২ মাস) create
  4. 4) দরকার হলে Opening balance journal entry দিন
  5. 5) প্রথম transaction draft করে test post দিন
  6. 6) Trial Balance দেখে debit-credit match নিশ্চিত করুন

Part D: Opening balance সেট করা

আগের সিস্টেম থেকে balance আনতে চাইলে Journal entries ব্যবহার করুন। উদাহরণ: Cash 500,000 debit, Owner Capital 500,000 credit। Save draft করে Post করুন।

Part E: দৈনন্দিন operation flow (প্রতিদিন)

Menu shortcut: Finance sidebar-এ এখন আলাদা `AP invoices`, `AP payments`, `AR invoices`, `AR receipts` link আছে।

১) Supplier purchase: Finance → AP Invoices → Add → lines দিন → Save → Post

২) Supplier payment: Finance → AP Payments → Add → Save → Post

৩) Customer sale: Finance → AR Invoices → Add → lines দিন → Save → Post

৪) Customer receipt: Finance → AR Receipts → Add → Save → Post

৫) Cash/Bank transfer: Finance → Cash transactions → Add → Save → Post

Part F: সপ্তাহ/মাস ভিত্তিক কাজ

  • - সপ্তাহে ১ বার: GL report দেখে unusual entry check করুন
  • - মাস শেষে: Trial Balance মিলান
  • - Profit & Loss দেখে expense বেশি কিনা দেখুন
  • - Balance Sheet দেখে cash, receivable, payable health check করুন
  • - Budget vs Actual দেখে plan vs actual gap বের করুন

Part H: Complete demo scenario (এক মাসে কী হলো)

  1. 1) রাহাত 500,000 টাকা capital দিলেন (Journal)
  2. 2) Supplier থেকে পণ্য কিনলেন 120,000 + VAT (AP Invoice Posted)
  3. 3) Supplier-কে 60,000 payment দিলেন (AP Payment Posted)
  4. 4) Customer-কে বিক্রি 200,000 + VAT (AR Invoice Posted)
  5. 5) Customer 150,000 receive হলো (AR Receipt Posted)
  6. 6) মাস শেষে Profit & Loss এ লাভ, Balance Sheet এ cash/receivable/payable সব দেখা গেল

Part I: Troubleshooting (common ভুল)

  • - Trial balance mismatch: কোনো draft/posting অসম্পূর্ণ কিনা দেখুন
  • - AP/AR post error: expense/ap/ar/tax account mapping ফর্মে দেওয়া আছে কিনা দেখুন
  • - Currency issue: exchange rate আছে কিনা check করুন
  • - Line total ভুল: qty ও unit price ঠিক আছে কিনা, deleted row tick আছে কিনা check করুন
  • - Wrong period: posting date সঠিক fiscal period এর মধ্যে কিনা দেখুন
